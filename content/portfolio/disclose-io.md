## About The Disclose.io Project

The Disclose.io Project is a collaborative and vendor-agnostic initiative to standardize best practices around safe harbor for good-faith security research. The project helps create a healthy and sustainable relationship between security researchers and the organizations they're trying to help.

## Key Features

- Standardized vulnerability disclosure policy templates
- Safe harbor clauses to protect security researchers
- Community-driven database of security contacts and programs
- Tools to help organizations implement secure disclosure practices

## Impact

The Disclose.io Project has helped thousands of organizations implement responsible disclosure programs, creating safer environments for security researchers to report vulnerabilities without fear of legal repercussions. The project continues to advocate for better security practices and clearer legal frameworks around security research.
